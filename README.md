<b>Instalação:</b>
<br>É necessário ter o nodejs, clone o projeto, entre no diretório da app pelo terminal e digite `npm install` (para instalar todas dependências).
<br>
<b>Rodar o projeto:</b>
<br>Digite node app e abra seu navagador na url: localhost:3000
<br>
<br>
<b>Dependências</b>
<ul>
    <li>body-parser</li>
    <li>consign</li>
    <li>ejs</li>
    <li>express</li>
    <li>express-validator</li>
    <li>socket.io</li>
</ul>
